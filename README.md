
# NEMORY — Живий простір памʼяті

**NEMORY** — це веб-застосунок, створений для внутрішньої тиші, глибини і фіксації станів.

## Що в ньому є:
- Інтерактивна памʼять (запис думок, станів)
- Візуалізація пелюсток памʼяті
- Атмосферний фоновий звук
- Мʼякий інтерфейс і досвід тиші

## Як запустити:
1. Завантажте `index.html` та відкрийте в браузері
2. Або розгорніть через GitHub Pages

## Авторство
Створено з любовʼю в NEMORY.
