# NEMORY — Cyberpunk PWA

Welcome to **NEMORY**, a cyberpunk-inspired Progressive Web App (PWA) experience that merges memory, identity, and atmosphere into one glowing interface.

## Features

- ⚡ Neon-styled animated intro (`intro.html`)
- 🎵 Custom startup sound
- ⚙️ Glitchy animated UI with cyberpunk theme
- 📱 Installable as a mobile/web app (PWA-ready)
- 💾 Offline support via Service Worker
- ✨ Memory logging system (`nemory.html`)

## How to Use

1. Open [`intro.html`](intro.html) to launch the experience.
2. After the animated sequence, you're redirected to the main interface.
3. Explore NEMORY. Click, type, remember.

## Deployment

You can host this project on:
- [GitHub Pages](https://pages.github.com/)
- [Netlify](https://www.netlify.com/)
- [Vercel](https://vercel.com/)

## Install as App

Open in Chrome or Android browser → "Add to Home Screen" to install as a PWA.
